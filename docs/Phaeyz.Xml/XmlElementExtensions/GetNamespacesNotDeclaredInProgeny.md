# XmlElementExtensions.GetNamespacesNotDeclaredInProgeny method

Gets a set of all namespaces which are referenced but not declared in the element or its descendants.

```csharp
public static HashSet<XmlPrefixedNamespace> GetNamespacesNotDeclaredInProgeny(this XmlElement @this)
```

| parameter | description |
| --- | --- |
| this | The element to scan for undeclared namespaces. |

## Return Value

A set of all namespaces which are referenced but not declared when referenced.

## Remarks

This does not perform validation of the XML document such as redefinition of namespaces using the same prefix, or elements which use a prefix but the namespace URI is wrong. A namespace is considered declared as long as a prefix is defined (including the default prefix).

# XmlElementExtensions.EnsureNamespaceDeclared method

Ensures there is a namespace declaration for the given namespace name and prefix.

```csharp
public static XmlElement EnsureNamespaceDeclared(this XmlElement @this, 
    bool declareEvenIfInherited, XmlPrefixedNamespace prefixedNamespace)
```

| parameter | description |
| --- | --- |
| this | The XmlElement to ensure there is a namespace declaration. |
| declareEvenIfInherited | If `true`, a namespace declaration will be added even if the namespace is inherited from a parent element. If `false`, a namespace declaration will not be added if the namespace is inherited from a parent element. |
| prefixedNamespace | The namespace to ensure is declared. |

## Return Value

The XmlElement passed in as input.

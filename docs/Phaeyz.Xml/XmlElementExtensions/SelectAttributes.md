# XmlElementExtensions.SelectAttributes method

Selects attributes of the current XmlElement.

```csharp
public static IEnumerable<XmlAttribute> SelectAttributes(this XmlElement @this, 
    string? namespaceUri = null, string? localName = null)
```

| parameter | description |

| localName | Optionally filters all attributes to the specified local name. Specify `null` for all local names. The default is `null` if unspecified. |
